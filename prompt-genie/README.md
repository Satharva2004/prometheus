# Prometheus

## Project info

This is a React project built with Vite, TypeScript, and Tailwind CSS.

## How to run locally

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Steps

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd prometheus

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

You can deploy this project to Vercel, Netlify, or any other static site hosting provider.
